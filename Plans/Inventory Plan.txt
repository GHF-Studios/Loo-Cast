- Item
	- Represents a specific item instance
	- Contains a Count
	- Contains an ItemData
	- Contains all of the item's gameplay logic
	- Can Instantiate and Destroy itself's physical representation (ItemDrop)
		- ItemDrop CreateItemDrop()
		- void DestroyItemDrop(ItemDrop itemDrop)

- ItemData : ScriptableObject
	- Contains an items default data
	
- ItemDataSet : ScriptableObject
	- Contains all ItemDatas for easy lookup
	
- ItemObject : MonoBehaviour
	- Represents a specific item instance
	- Holds an Item
	
- ItemContainer : ScriptableObject
	- Holds a list of Slots
	
- ItemSlot
	- Holds an Item
	
Examples:

- WeaponItem
- ModuleItem
- ResourceItem
