# Modding Status

Current implementation status summary.

## What Exists

- Core and base mods are integrated into the workspace.
- Runtime/build paths support mod crates built as dynamic libraries.
- Init hook convention exists (`init_api` feature + exported init symbols).

## What Is Not Stable Yet

- External mod ecosystem and discovery contracts are not complete.
- Dependency/conflict metadata format is not fully locked.
- Tooling for robust third-party mod packaging is incomplete.

## Use This File As

- A status note for implementation reality.
- Not a frozen target-spec for future modding architecture.
