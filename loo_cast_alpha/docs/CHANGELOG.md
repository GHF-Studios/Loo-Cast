# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and published game/contract versions use [Semantic Versioning](https://semver.org/).

Pre-`1.0.0` versions are initial development. Public API stability is not guaranteed until `1.0.0`; see
`CONTRACTS.md` for the project-specific milestone meanings.

## [Unreleased]

### Added

- Alpha workspace execution rails.
- `xtask` build, package, run, setup, and audit entrypoints.
- Phase milestone, issue, gate, and pull request templates.

## [0.5.0] - TBD

Planned alpha milestone. The feature framework, mod/API shape, and execution rails are coherent enough for sustained
feature and gameplay development.

## [0.9.0] - TBD

Planned beta milestone. Feature/framework direction is mostly locked, and work shifts toward bug fixing, refinement,
compatibility, packaging, and release hardening.

## [1.0.0] - TBD

Planned stable commercial release baseline. Public API compatibility rules become stricter, and the project can move
toward its finalized open-source/licensing structure.
