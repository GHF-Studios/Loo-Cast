The USF Contract defines foundational simulation principles and contract-level structure in core_engine.
It defines both capability typing and [[Scale]] as core constituents.
A [[Scale]] includes capability types, metric types, and phenomena types used to describe what is meaningful at that
scale.
Scale compatibility declarations are defined by the [[Scale Contract]].
The default runtime counterpart is the singleton-like [[USF Runtime]] implementation in core_mod.
