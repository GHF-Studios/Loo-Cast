Scale is a complex util/support type defined inside the [[USF Contract]].
It acts as metadata and execution-context structure for simulation semantics at a given scale.
A scale specifies capability types, metric types, phenomena types, and one phenomena realizer type, and participates in
coordinating capability behavior through [[Scaled Capability Channel]]s and the [[Capability Runtime]].
