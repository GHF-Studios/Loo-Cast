This subtree is an exploratory ontology engine embedded in markdown.
It is intentionally cordoned off from the primary glossary flow.

Scope of this first iteration:

1. Define a node-centric ontology kernel.
2. Define conditional metadata activation as a dependency graph.
3. Define namespaced edge taxonomy and morphism policy.
4. Demonstrate projection-based books over shared nodes.
5. Provide example paradigm, solver, transform, and morphism nodes.
6. Drive broad coverage across all major simulation domains.

Cordon rules:

- Everything under `experimental_simulation_ontology/` is non-contractual and may break.
- No existing glossary file outside this subtree is modified except index linking.
- Projection books are traversal guides only. Node content lives only in `Nodes/`.
