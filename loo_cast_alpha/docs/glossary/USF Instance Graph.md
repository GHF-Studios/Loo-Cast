The USF Instance Graph is the active structured set of instantiated USF objects and their relations.
It is organized as a 71-scale stack of canonical simulation slices through [[Scale]].
In the runtime substrate, these slices are simulated in parallel and composed through capability outputs.
This graph is constrained by contract-level invariants including [[Scale Realizer Cardinality]].
