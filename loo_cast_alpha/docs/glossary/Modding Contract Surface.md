The Modding Contract Surface defines how mods declare integration points, lifecycle hooks, compatibility expectations,
and extension boundaries across the runtime stack.
It is one contract family within the [[Contract Surface]] and composes with the [[Capability Contract Surface]].
Through this family, mods can introduce new capabilities and extend existing capability channels.
