# Rhai-First Std Space (Scaffold)

This folder is reserved for script modules that exist primarily for Rhai authoring ergonomics.

Use this for:

- Rhai-only utility functions and helper abstractions,
- higher-level wrappers over raw bridge calls,
- reusable script patterns that should not live in startup hook suites.

Keep startup suites focused on integration coverage and examples.
Keep reusable script libraries here.
